# Rule-Based AI Chatbot 🤖

## Project Overview

This project is a simple Rule-Based AI Chatbot developed as part of the DecodeLabs Artificial Intelligence Internship Program. The chatbot interacts with users by responding to predefined questions using if-else conditions and a continuous loop.

The purpose of this project is to understand the fundamentals of Artificial Intelligence, decision-making logic, control flow, and user interaction through programming.

## Features

* Handles greetings such as Hello, Hi, and Hey.
* Responds to predefined questions.
* Uses if-else statements for decision making.
* Runs continuously until the user chooses to exit.
* Handles exit commands such as Bye, Exit, and Quit.
* Includes a help menu displaying available commands.
* Provides basic information about AI, Python, Chatbots, Machine Learning, and Internships.
* Demonstrates nested conditions through user mood responses.

## Technologies Used

* Python
* Control Flow Statements
* Conditional Logic (if-elif-else)
* Loops (while loop)

## How It Works

1. The chatbot waits for user input.
2. It compares the input with predefined questions.
3. If a match is found, an appropriate response is displayed.
4. If no match is found, the chatbot informs the user that the question is not recognized.
5. The chatbot continues running until the user enters an exit command.

## Sample Questions

* How are you?
* What is your name?
* Who made you?
* What is AI?
* What is Python?
* What is Internship?
* What is a Chatbot?
* What is Machine Learning?
* What can you do?
